# 手順と必要書類
ハンブルクでの賃貸アパート探しは**非常に競争が激しい**ですが、しっかり戦略を立てれば、良い物件に巡り合うチャンスを高められます。以下に、実践的な**おすすめの手順と戦略**を日本語で分かりやすくまとめます。

---

## ✅ ステップ別アパート探し戦略【ハンブルク版】

---

### 🧭 **Step 1：希望条件の整理**

まず、自分の条件を明確にしましょう：

| 項目     | 例                                       |
| ------ | --------------------------------------- |
| 家賃上限   | €800〜1,200（暖房費込みかどうかも確認）                |
| エリア希望  | Eimsbüttel / Altona / Barmbek / Horn など |
| 広さ・部屋数 | 2 Zimmer（1LDK相当）以上など                    |
| 入居希望日  | すぐ入居可？それとも◯月以降？                         |
| その他    | ペット可、バルコニー、地下室など                        |

---

### 🔍 **Step 2：物件探し（ポータルサイト利用）**

#### 主なサイト（必ず毎日チェック）：

* [Immobilienscout24](https://www.immobilienscout24.de)
* [Immonet](https://www.immonet.de)
* [Immowelt](https://www.immowelt.de)
* [WG-Gesucht](https://www.wg-gesucht.de)（一人暮らし/カップル向け物件あり）
* [eBay Kleinanzeigen](https://www.kleinanzeigen.de)（詐欺注意）

📌 **TIP**:
検索フィルターで「**provisionsfrei（仲介手数料なし）**」にチェックを入れるとよいです。

---

### 📝 **Step 3：応募書類の準備（最重要！）**

ドイツでは**スピード＆信用が命**。あらかじめ以下をPDFでまとめておくと即応募可能。

#### 💼 応募書類セット「Mietbewerbungsmappe」に入れるもの：

1. ✅ **Schufa（信用情報）**
2. ✅ **給与明細（直近3ヶ月）**
3. ✅ **雇用証明書（Arbeitsvertrag）**
4. ✅ **身分証コピー（パスポート＋Meldebescheinigung）**
5. ✅ **前家主からの推薦状（Vorvermieterbescheinigung）**
6. ✅ **自己紹介文（ドイツ語）**
   　→ 例：「夫婦共に公務員で非喫煙者、静かで長期居住希望」など

📁 すべてを**1つのPDFファイルにまとめて**、物件応募時に即送れるようにしておくと◎

---

### 💬 **Step 4：内見・応募の対応戦略**

* 🔔 **内見予約は即レス**：「見るだけ予約」でも即申し込む意思を見せると効果的。
* 👔 **内見時は好印象を意識**（礼儀正しく、整理された服装）
* ✉️ **内見後すぐに「Danke」メール＋応募書類再送**でアピール
* 📄 **カバーレターを簡潔に添える**（自己紹介＋希望理由）

---

### 🧠 **Step 5：複数並行応募 & 詐欺対策**

#### ✅ 同時に複数応募してOK

* 特にハンブルクでは**競争率が高く、1件で決まることは稀**。
* 10件〜20件は並行応募が普通です。

#### ⚠️ 詐欺注意ポイント

| 危険サイン         | 内容                 |
| ------------- | ------------------ |
| 家賃が相場より明らかに安い | 例：Eppendorfで€500など |
| 内見なしで前払いを要求   | 「鍵を送るから送金して」→ 詐欺   |
| メールで英語だけでやり取り | 不自然な英語メール＝要注意      |

---

## 🏁 最後に：おすすめのエリア（初めての方向け）

| エリア名           | 特徴                    |
| -------------- | --------------------- |
| **Eimsbüttel** | 若者と家族に人気、カフェ多め、便利     |
| **Altona**     | 駅近くて中心部に出やすく、おしゃれ     |
| **Barmbek**    | 家賃は比較的手頃、治安も安定        |
| **Horn**       | 物件多め、緑が多い、最近人気上昇中     |
| **Wandsbek**   | 落ち着いた住宅街＋ショッピングセンターあり |

---

## 📦 まとめチェックリスト

* ✅ 希望条件を整理済み
* ✅ 書類はPDFで揃えてある
* ✅ 物件サイトをブックマーク
* ✅ Schufaなど信用情報OK
* ✅ ドイツ語の自己紹介文も準備
* ✅ 詐欺のパターンも把握している

# Vorvermieterbescheinigungはあるとベター
とても良い質問です！

---

## ✅ **Vorvermieterbescheinigung（前家主の推薦状）について**

### ● 必要か？

→ **ほぼ必須ではないけど、あると有利！**

### 💡 どういうときに役立つ？

* 他の応募者と差をつけたいとき
* 信用スコア（Schufa）が薄い／まだドイツに来たばかりの人
* 書類審査だけで選ばれるような物件（人気物件）

### 📄 書類に書かれること：

* 家賃支払いの遅れがなかったか
* トラブルを起こしていないか
* 契約違反などがなかったか

> ドイツでは「この人は良いテナントだったよ」と第三者から証明されることがとても重視されます。

### ✅ 書類テンプレが必要なら用意できます！

---

## ⚠️ **Kleinanzeigen（旧eBay Kleinanzeigen）について**

### ● 安全か？

→ **リスクは高め、でも注意すれば使える**

### 🌟 メリット：

* 家主と直接やり取りできる（手数料がない）
* 他のサイトより早く・柔軟に決まることも
* 短期や個人契約向け物件が多い

### ⚠️ デメリット／詐欺の危険サイン：

| 詐欺サイン                | 内容                   |
| -------------------- | -------------------- |
| 写真が不自然に綺麗 or ストックっぽい | 偽物の可能性あり             |
| 英語のみでやり取り            | 「海外にいるが鍵は送る」など要注意    |
| すぐ送金を求める             | 「PayPalなら安全」と言いながら詐欺 |
| 家賃が異常に安い             | 中心部で€400〜500などはほぼ嘘   |

> 🔒 **ルール**：**内見せずにお金を送らない。** これを守れば9割防げます。

---

## ✅ Kleinanzeigenを安全に使うコツ

1. **必ず内見（Besichtigung）をすること**
2. **家主の連絡先や身元を確認する**
3. **書類や契約内容をきちんと確認（口約束NG）**
4. **可能ならドイツ語でやり取り（不自然な英語は注意）**

---

## ✋ 結論まとめ

| 質問                            | 回答                              |
| ----------------------------- | ------------------------------- |
| Vorvermieterbescheinigungは必要？ | **なくてもよいが、あると明確に有利！**           |
| Kleinanzeigenは怖い？             | **詐欺リスクあり。ただし注意すれば安全に使える手段の一つ** |

---

必要なら：

* Vorvermieterbescheinigungのテンプレート（英語＆ドイツ語）
* Kleinanzeigenのやり取り文例（ドイツ語）

# immobilienscout24でschufa信用チェックも作成した
はい、Immobilienscout24経由で取得できる **SCHUFAボナitätsCheck（信用チェック書類）** は、**正式かつ公式に認められているもの**です。ドイツ国内の賃貸契約において、広く受け入れられており、大家や不動産会社に提出するのにまったく問題ありません。

---

### ✅ Immobilienscout24経由のSCHUFAの特徴

| 項目   | 内容                                                                   |
| ---- | -------------------------------------------------------------------- |
| 発行元  | SCHUFA（Schutzgemeinschaft für allgemeine Kreditsicherung）※ドイツの信用情報機関 |
| 提供形態 | ダウンロード（PDF） + 郵送（希望すれば）                                              |
| 利用目的 | 賃貸申込み、就職、携帯契約など                                                      |
| 言語   | ドイツ語（標準）                                                             |
| 有効性  | ドイツ全土で法的に有効、賃貸契約でも一般的に通用する                                           |

---

### 🎯 賃貸申込みで通用する？

**はい、ほぼすべての大家・不動産会社で通用します。**
Immobilienscout24経由のSCHUFAは、一般的に「Bonitätsauskunft（信用証明）」として正式に受け取ってもらえます。

---

### 💰 費用の目安（参考）

* Immobilienscout24経由：約**29.95ユーロ**
* 直接SCHUFA公式から取得する価格とほぼ同じ

---

### 💡 補足：こんなときに便利

* 物件応募フォームで「SCHUFAをアップロードしてください」と書かれている
* 書類の信頼性に不安があるとき（PDFに公式ロゴ＆チェックコードあり）

---

### ✅ まとめ

> **Immobilienscout24経由のSCHUFA信用チェックは、公式・信頼性のある書類であり、賃貸応募に安心して使えます。**

必要であれば、取得ページの案内や、実際の書類の見本説明もできますので、お気軽に聞いてくださいね。

# 住宅協同組合
ハンブルクでは、近年、住宅不足への対応として、さまざまな地区で積極的な住宅建設プロジェクトが進行中です。以下に、特に注目される地区と活動的な住宅協同組合（Wohnungsgenossenschaften）をご紹介します。

---

## 🏗️ ハンブルクで注目の住宅建設地区

### 1. **オーバービルヴェルダー（Oberbillwerder）**

ハンブルクの105番目の新しい地区として計画されており、最大7,000戸の住宅が建設される予定です。このプロジェクトには、学校、大学キャンパス、公園なども含まれており、約15,000人の住民が見込まれています。 ([Haufe.de News und Fachwissen][1])

### 2. **ペルゴレンフィアテル（Pergolenviertel）**

ハンブルク北部のヴィンターフーデ地区に位置し、約1,700戸の住宅が建設されています。多様な住宅形態が特徴で、都市と自然が調和した新しい街区です。&#x20;

### 3. **ノイグラーベン＝フィッシュベック（Neugraben-Fischbek）**

南部のこの地区では、「自然と共に暮らす」をテーマに、以下の3つのプロジェクトが進行中です：

* **フォーゲルカンプ・ノイグラーベン（Vogelkamp Neugraben）**：約1,700戸の住宅が計画されています。

* **フィッシュベッカー・ハイドブルック（Fischbeker Heidbrook）**：約1,250戸の住宅が建設され、自然保護区に隣接しています。

* **フィッシュベッカー・レーゼン（Fischbeker Reethen）**：約2,300戸の住宅が計画されており、持続可能な都市開発が進められています。 ([Wikipedia][2])

---

## 🏘️ 活動的な住宅協同組合（Wohnungsgenossenschaften）

### 1. **バウフェライン・デア・エルプゲマインデン（Bauverein der Elbgemeinden eG, BVE）**

ハンブルク最大の住宅協同組合で、約14,500戸の住宅と23,000人以上の会員を有しています。持続可能な住宅供給とコミュニティ形成に注力しています。 ([bve.de][3])

### 2. **ハンザ・バウゲノッセンシャフト（HANSA Baugenossenschaft eG）**

約10,000戸の住宅を所有し、特にハンブルク中央部に多くの物件があります。現在、ハーフェンシティのシュトラントカイで81戸、ローブルッゲで70戸の新築プロジェクトを進行中です。 ([Wikipedia][4])

### 3. **シュフツィンメラー（Allgemeine Deutsche Schiffszimmerer-Genossenschaft eG）**

ハンブルク最古の住宅協同組合で、魅力的な住宅提供と地域社会の発展に貢献しています。 ([schiffszimmerer.de][5])

### 4. **バウゲノッセンシャフト・メスターカンプ（Baugenossenschaft Mesterkamp）**

バルムベック＝ズュート地区で、約40戸の公共支援住宅を建設中です。「車なし、持続可能、世代を超えた共生」をモットーにしています。 ([Baugenossenschaft Mesterkamp][6])

### 5. **BDS ハンブルク（BDS Hamburg）**

ハンブルク北部のグロース・ボルステルやファルムゼン、ハーフェンシティなどで新築プロジェクトを展開しています。地域に根ざした住宅供給を行っています。&#x20;

---

これらの地区や住宅協同組合は、ハンブルクでの住宅探しにおいて注目すべき存在です。特に、持続可能性やコミュニティ形成を重視する方には、住宅協同組合の物件が適しているかもしれません。具体的な物件情報や申込方法については、各協同組合の公式ウェブサイトをご参照ください。

## 🔍 入居を希望する際のポイント

* **登録とプロフィール作成**：多くの協同組合では、Immomioなどのプラットフォームへの登録と、希望条件を含むプロフィールの作成が求められます。これにより、希望に合った物件の情報を受け取ることができます。([bgfg.de][3])

* **定期的な確認**：空き物件の情報は頻繁に更新されるため、定期的に各協同組合のウェブサイトを確認することをおすすめします。

* **必要書類の準備**：入居申請時には、収入証明書や身分証明書などの書類が必要となる場合があります。事前に準備しておくとスムーズです。
