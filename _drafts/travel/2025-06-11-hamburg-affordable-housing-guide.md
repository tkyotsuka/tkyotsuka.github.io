# ハンブルクで安い住まいを見つけるためのガイド：加入すべき団体と組織

ハンブルクは、ドイツで2番目に大きい都市であり、魅力的な都市ですが、住宅市場は非常に競争が激しく、手頃な価格の住まいを見つけることは難しい場合があります。この記事では、ハンブルクで安い住まいを見つけるために加入すべき主要な団体や組織について詳しく説明します。

## 住宅協同組合（Wohnungsgenossenschaften）

### 住宅協同組合とは？

住宅協同組合（Wohnungsgenossenschaft）は、ドイツで長い歴史を持つ住宅提供の形態です。協同組合のメンバーになることで、手頃な価格の住宅へのアクセスが可能になります。実際、ハンブルクで最初の住宅協同組合は1862年に設立されました。

### 仕組み

1. **会員制**: 協同組合に加入するには、出資金（Genossenschaftsanteil）を支払う必要があります。これは通常、数百ユーロから数千ユーロの範囲です。
2. **非営利**: 商業的な住宅会社と異なり、住宅協同組合は利益を追求せず、会員に手頃な価格の住宅を提供することを目的としています。
3. **安定した家賃**: 協同組合の住宅は通常、市場価格よりも安く、家賃の急激な上昇からも保護されています。
4. **長期的な住居権**: 会員は、ルールに従っている限り、長期間住み続ける権利があります。
5. **共同決定**: 会員は協同組合の意思決定プロセスに参加する権利があります。

### ハンブルクの主要な住宅協同組合

ハンブルクには約30の住宅協同組合があり、合計で約135,000戸の住宅を管理し、約230,000人の会員とその家族に住居を提供しています。主要な協同組合には以下が含まれます：

1. **Baugenossenschaft freier Gewerkschafter eG (BGFG)**
   - [公式ウェブサイト](https://www.bgfg.de/)
   - 約7,500戸の住宅を管理
   - 出資金：約500〜1,000ユーロ

2. **Baugenossenschaft Hamburger Wohnen eG**
   - [公式ウェブサイト](https://www.hamburger-wohnen.de/)
   - 約4,500戸の住宅を管理
   - 出資金：約500〜800ユーロ

3. **Allgemeine Deutsche Schiffszimmerer-Genossenschaft eG**
   - [公式ウェブサイト](https://www.schiffszimmerer.de/)
   - 約8,000戸の住宅を管理
   - 出資金：約500〜1,000ユーロ

4. **HANSA Baugenossenschaft eG**
   - [公式ウェブサイト](https://www.hansa-baugenossenschaft.de/)
   - 約9,800戸の住宅を管理
   - 出資金：約500〜1,000ユーロ

5. **Wohnungsgenossenschaft von 1904 e.G.**
   - [公式ウェブサイト](https://www.wg1904.de/)
   - 約2,300戸の住宅を管理
   - 出資金：約300〜600ユーロ

### 会員数あたりのアパート数比率

住宅協同組合を選ぶ際に重要な指標の一つが、会員数あたりのアパート数（アパート・会員比率）です。この比率が高いほど、理論的には会員が住宅を得られる確率が高くなります。

**ハンブルク全体の平均**：
- 約30の住宅協同組合で合計約130,000〜135,000戸の住宅
- 合計約230,000人の会員
- **平均比率：約0.59戸/会員**（100会員あたり59戸）

**主要協同組合の比率**：

1. **BGFG (Baugenossenschaft freier Gewerkschafter)**
   - 約7,600戸の住宅
   - 11,000人以上の会員
   - **比率：約0.69戸/会員**（100会員あたり69戸）
   - 全体平均より高い比率で、比較的有利

2. **HANSA Baugenossenschaft**
   - 10,000戸以上の住宅
   - 会員数の具体的な情報は公開されていませんが、規模の大きい協同組合です

3. **小規模協同組合の可能性**
   - 「Wohnungsgenossenschaft von 1904 e.G.」などの小規模協同組合（約2,300戸）は、会員数も少ない可能性があり、比率が良い場合があります

**比率の意味**：
- **高い比率（0.7以上）**: 会員あたりの住宅数が多く、比較的短期間で住宅を得られる可能性が高い
- **平均的な比率（0.5〜0.7）**: 標準的な待機期間が必要
- **低い比率（0.5未満）**: 会員あたりの住宅数が少なく、待機期間が長くなる可能性が高い

**戦略的アドバイス**：
- 可能であれば、BGFGのような比率の高い協同組合と、小規模だが専門的な協同組合の両方に加入することで、住宅を得る確率を高められます
- 人気エリアでは待機期間が長くなる傾向があるため、より郊外のエリアも検討することで、待機期間を短縮できる可能性があります

### 住宅協同組合に加入するメリット

- 市場価格よりも安い家賃
- 家賃の安定性（急激な値上げがない）
- 高品質の住宅と良好なメンテナンス
- 強いコミュニティ意識と住民参加
- 長期的な住居の安全性

### 外国籍の方の加入資格

ハンブルクの住宅協同組合は一般的に、国籍に関する制限を設けていません。外国籍の方も以下の条件を満たせば加入できます：

1. **EU市民**：
   - EU市民は、ドイツ国民と同等の権利を持ち、特別な許可なく住宅協同組合に加入できます
   - 通常の申請手続きと出資金の支払いが必要です

2. **非EU市民（日本国籍を含む）**：
   - 有効な滞在許可証（Aufenthaltstitel）が必要です
   - 長期滞在予定であることが望ましいです（少なくとも1年以上）
   - その他の要件はドイツ国民と同じです（出資金の支払い、申請書の提出など）

3. **すべての外国籍の方に共通する要件**：
   - 基本的なドイツ語能力（少なくとも契約内容を理解できるレベル）
   - ドイツの銀行口座（家賃の自動引き落としのため）
   - 安定した収入の証明（給与明細や雇用契約書など）

住宅協同組合によっては、外国籍の方向けの多言語サポートを提供している場合もあります。例えば、BGFGなどの大手協同組合では英語での対応が可能な場合があります。

### 住宅協同組合に加入する方法

1. 興味のある協同組合のウェブサイトを訪問し、会員申請フォームをダウンロードするか、オンラインで申請します。
2. 必要な出資金を支払います（協同組合によって異なります）。
3. 会員になると、空き物件の待機リストに登録されます。
4. 待機期間は協同組合や地域によって異なりますが、数ヶ月から数年かかる場合があります。

### 注意点

- 多くの協同組合では、ドイツ語の知識が必要です。
- 人気のある地域では待機期間が長くなる場合があります。
- 出資金は退会時に返還されますが、通常は3ヶ月の通知期間が必要です。

## 学生寮（Studierendenwerk Hamburg）

### Studierendenwerk Hamburgとは？

Studierendenwerk Hamburgは、ハンブルクの学生と研修生に手頃な価格の住居を提供する公的機関です。25の住宅施設で約4,390の住居を提供しています。

### 特徴

1. **手頃な価格**: 市場価格よりも大幅に安い家賃（光熱費込み）
2. **様々な住居タイプ**: 共有アパート（WG）の個室、1人用アパート、2人用アパートなど
3. **便利な立地**: 多くの住居は大学やトレーニング施設の近くにあります
4. **国際的な環境**: ドイツ人学生と留学生の両方が住んでいます

### 申し込み方法

1. [オンライン申請ポータル](https://www.stwhh.de/en/accommodation-for-students-and-trainees/online-application)を通じて申し込みます。
2. 希望する入居日の3ヶ月前から申し込みが可能です。
3. 学生または研修生であることの証明が必要です。
4. デポジット（敷金）が必要です：
   - 月額家賃300ユーロまで = 400ユーロのデポジット
   - 月額家賃400ユーロまで = 500ユーロのデポジット
   - 月額家賃450ユーロまで = 600ユーロのデポジット
   - 月額家賃600ユーロまで = 700ユーロのデポジット
   - 月額家賃601ユーロ以上 = 800ユーロのデポジット

### 注意点

- 人気が高いため、早めに申し込むことをお勧めします。
- 特定の住居施設を希望する場合は、待機期間が長くなる可能性があります。
- すべての住居施設に申し込むと、待機期間が短くなる可能性があります。

## SAGA - ハンブルク市営住宅会社

### SAGAとは？

SAGA（Siedlungs-Aktiengesellschaft Hamburg）は、ドイツ最大の公営住宅会社であり、ハンブルク市が所有しています。手頃な価格の住宅を提供し、地域開発に大きく貢献しています。

### 特徴

1. **大規模な住宅ストック**: ハンブルク全域に多数の住宅を所有・管理
2. **手頃な価格**: 市場価格よりも安い家賃
3. **様々な住居タイプ**: 単身者向けから家族向けまで、様々なサイズの住居
4. **地域に根ざしたサービス**: 地域ごとに事務所があり、住民に直接サービスを提供

### 申し込み方法

1. [SAGA公式ウェブサイト](https://www.saga.hamburg)で登録します。
2. immomioというパートナープラットフォームを通じて物件を検索・申し込みができます。
3. 必要書類（収入証明、SCHUFA信用情報など）を提出します。

### 注意点

- 需要が高いため、申し込みから入居までに時間がかかる場合があります。
- 収入制限がある場合があります（特に補助付き住宅の場合）。
- ドイツ語の知識が必要です。

## その他の住宅オプション

### 1. 住宅協会（Wohnungsbauvereine）

住宅協同組合と似ていますが、組織構造が異なります。会員になることで、手頃な価格の住宅へのアクセスが可能になります。

### 2. 社会住宅（Sozialwohnungen）

低所得者向けの補助付き住宅です。申し込むには、住宅資格証明書（Wohnberechtigungsschein、通称WBS）が必要です。

### 3. シェアハウス（Wohngemeinschaft、通称WG）

特に学生や若い専門家の間で人気のある住居形態です。個室を借り、キッチンやバスルームなどの共用スペースを他の住人と共有します。

### 4. 一時的な住居オプション

- **ゲストハウス**: 短期滞在向け
- **サブレット（Zwischenmiete）**: 一時的に不在の住人から部屋を借りる
- **一時的な学生寮**: 学期間の短期滞在向け

## 暖房費用の目安

ハンブルクでは、多くの物件で暖房費（Heizkosten）が家賃に含まれていません。これは「Kaltmiete」（冷家賃）と呼ばれる基本家賃に加えて支払う必要がある重要な追加コストです。

### 月額暖房費の目安（70㎡のアパートの場合）

ドイツの暖房費用指標（Heizspiegel 2024）によると、70㎡のアパートの平均的な月額暖房費は以下の通りです：

| 暖房タイプ | 2023年平均月額 | 2024年予測月額 | 建物の築年数による変動 |
|---------|------------|------------|-----------------|
| ガス暖房（Erdgas） | 約111ユーロ | 約84ユーロ | 築1977年以前：約119ユーロ<br>築2002年以降：約70ユーロ |
| 石油暖房（Heizöl） | 約95ユーロ | 約91ユーロ | 築1977年以前：約99ユーロ<br>築2002年以降：約58ユーロ |
| 地域暖房（Fernwärme） | 約92ユーロ | 約111ユーロ | 築1977年以前：約102ユーロ<br>築2002年以降：約61ユーロ |
| ヒートポンプ（Wärmepumpe） | 約76ユーロ | 約63ユーロ | 築1977年以前：約81ユーロ<br>築2002年以降：約48ユーロ |
| 木質ペレット（Holzpellets） | 約70ユーロ | 約65ユーロ | 築1977年以前：約88ユーロ<br>築2002年以降：約53ユーロ |

### 暖房費用に影響する要因

1. **建物の築年数と断熱性能**：
   - 1977年以前に建てられた建物は、暖房費が最も高い傾向があります
   - 2002年以降に建てられた建物は、1977年以前の建物と比較して約40-42%暖房費が安くなります

2. **暖房システムの種類**：
   - ハンブルクでは地域暖房（Fernwärme）が一般的ですが、ガス暖房も多く見られます
   - 最近の建物ではヒートポンプなどの効率的なシステムも増えています

3. **季節による変動**：
   - 冬季（11月〜3月）は暖房費が大幅に高くなります
   - 夏季はほとんど暖房を使用しないため、費用は最小限になります

### 予算計画のアドバイス

1. **年間予算の立て方**：
   - 月々の暖房費は変動するため、年間総額を12で割った金額を毎月確保しておくことをお勧めします
   - 多くの場合、毎月一定額の前払い（Abschlag）を支払い、年末に実際の使用量に基づいて精算します

2. **エネルギー効率の確認**：
   - 物件を選ぶ際は、エネルギー証明書（Energieausweis）をチェックし、エネルギー効率の高い物件を選ぶことで暖房費を抑えられます
   - エネルギー効率クラスがA+〜Cの物件は暖房費が比較的安くなります

3. **その他の光熱費も考慮**：
   - 暖房費に加えて、電気代（Strom）や水道代も別途かかることを忘れないでください
   - 電気代の目安：1人世帯で月30〜50ユーロ、2人世帯で60〜80ユーロ

## 申し込み方法と注意点

### 必要書類

1. **個人身分証明書またはパスポート**
2. **収入証明**（給与明細、雇用契約書など）
3. **SCHUFA信用情報**（ドイツの信用情報機関からの報告書）
4. **前の家主からの証明書**（Mietschuldenfreiheitsbescheinigung）
5. **住民登録証明書**（Meldebescheinigung）

### 申し込みのコツ

1. **早めに行動する**: 人気のある都市では、住宅市場は非常に競争が激しいです。
2. **複数のオプションを検討する**: 一つの組織だけでなく、複数の組織に申し込むことで、成功の可能性が高まります。
3. **ネットワークを活用する**: 友人、同僚、ソーシャルメディアグループなどを通じて情報を得ることも有効です。
4. **地域を柔軟に考える**: 中心部から少し離れた地域も検討すると、より手頃な価格の住居を見つけやすくなります。
5. **ドイツ語のサポートを得る**: 可能であれば、ドイツ語を話せる友人や同僚に助けを求めましょう。

ハンブルクで手頃な価格の住居を見つけることは難しい場合がありますが、住宅協同組合、Studierendenwerk Hamburg、SAGAなどの組織に加入することで、市場価格よりも安い住居へのアクセスが可能になります。早めに行動し、複数のオプションを検討することが重要です。また、ドイツ語の知識があると、プロセスがスムーズになります。
