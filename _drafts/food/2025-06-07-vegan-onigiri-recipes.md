# ビーガンおにぎりレシピ集：植物性素材だけで作る美味しいおにぎり

おにぎりは日本の伝統的な携帯食で、シンプルながらも無限のバリエーションが楽しめる万能食です。この記事では、動物性食品を使わないビーガン向けのおにぎりレシピをご紹介します。植物性素材だけでも、栄養バランスが良く、美味しいおにぎりを作ることができます。

## 基本のビーガンおにぎりの作り方

### 材料（基本の塩おにぎり 2個分）

- 温かいご飯 1合（約180g）
- 塩 小さじ1/4〜1/2（お好みで調整）
- 海苔 2枚
- 白いりごま 小さじ1（お好みで）

### 基本の作り方

1. **手の準備**: 清潔な手に少量の塩（分量外）をまぶし、水で軽く湿らせます。これにより、ご飯がくっつきにくくなります。

2. **ご飯の準備**: 温かいご飯に塩を全体に混ぜます。ご飯が熱い場合は少し冷ましてから作業すると良いでしょう。

3. **成形**: 
   - 手のひらにご飯を取り、軽く押さえながら三角形や俵型に成形します。
   - 中心を少しくぼませて具材を入れる場合は、ご飯を半分に分け、片方に具材をのせてからもう片方で覆い、全体を成形します。

4. **仕上げ**: 
   - 海苔を巻きます。全体を包むか、一部分だけ巻くかはお好みで。
   - 必要に応じて白いりごまをまぶします。

### ビーガンおにぎりのための基本調味料

- **塩**: 自然塩や岩塩がおすすめです。
- **醤油**: 通常の醤油はビーガンですが、製品によっては魚介エキスが含まれている場合があるので確認しましょう。
- **味噌**: 大豆と塩だけで作られた味噌を選びましょう。
- **梅干し**: 無添加の梅干しを選ぶと良いでしょう。
- **ごま油**: 香り付けに少量使うと風味が豊かになります。
- **昆布茶**: 魚介出汁の代わりに旨味を加えるのに最適です。

## 具材別レシピ

### 1. 梅と紫蘇のおにぎり

**材料（2個分）**:
- 温かいご飯 1合
- 梅干し 2個（種を取り除いて細かく刻む）
- 刻んだ紫蘇 6〜8枚
- 白いりごま 小さじ1

**作り方**:
1. ご飯に刻んだ梅と紫蘇を混ぜます。
2. 手に塩水をつけ、おにぎりを成形します。
3. 白いりごまをまぶし、海苔を巻きます。

### 2. 昆布の佃煮おにぎり

**材料（2個分）**:
- 温かいご飯 1合
- ビーガン昆布の佃煮 大さじ2（市販品または手作り）
- 白いりごま 小さじ1

**手作り昆布の佃煮**:
1. 乾燥昆布 20gを水で戻し、細切りにします。
2. 鍋に昆布、水 1/2カップ、醤油 大さじ2、砂糖 大さじ1、みりん 大さじ1を入れて煮詰めます。
3. 水分がほとんどなくなるまで煮詰めたら完成です。

**作り方**:
1. ご飯に昆布の佃煮を混ぜます。
2. 手に塩水をつけ、おにぎりを成形します。
3. 必要に応じて海苔を巻きます。

### 3. きのこの炒め煮おにぎり

**材料（2個分）**:
- 温かいご飯 1合
- しいたけ 2個
- しめじ 1/2パック
- 醤油 小さじ2
- みりん 小さじ1
- ごま油 小さじ1/2
- 刻みねぎ 適量

**作り方**:
1. きのこを細かく刻み、ごま油で炒めます。
2. 醤油とみりんを加えて軽く煮詰めます。
3. 火を止めて刻みねぎを加え、冷まします。
4. ご飯の中心にきのこの炒め煮を入れておにぎりを成形します。
5. 海苔を巻いて完成です。

### 4. 高菜漬けおにぎり

**材料（2個分）**:
- 温かいご飯 1合
- ビーガン高菜漬け 50g（魚介エキス不使用のもの）
- 白いりごま 小さじ1

**作り方**:
1. 高菜漬けを細かく刻みます。
2. ご飯に刻んだ高菜漬けを混ぜます。
3. 手に塩水をつけ、おにぎりを成形します。
4. 白いりごまをまぶし、海苔を巻きます。

### 5. 枝豆と塩昆布のおにぎり

**材料（2個分）**:
- 温かいご飯 1合
- 茹でた枝豆（さやから出したもの） 1/2カップ
- 塩昆布（魚介エキス不使用のもの） 大さじ1
- 白いりごま 小さじ1

**作り方**:
1. 枝豆を軽く潰します。
2. ご飯に潰した枝豆と塩昆布を混ぜます。
3. 手に塩水をつけ、おにぎりを成形します。
4. 白いりごまをまぶし、海苔を巻きます。

## 変わり種おにぎり

### 1. カレーおにぎり

**材料（2個分）**:
- 温かいご飯 1合
- ビーガンカレールウ 大さじ2（市販のビーガン対応品または手作り）
- 炒めた玉ねぎ 1/4個分
- 茹でたじゃがいも 1/4個（小さめに切る）
- 茹でたにんじん 1/4本（小さめに切る）
- 白いりごま 小さじ1

**作り方**:
1. カレールウと野菜を混ぜ、少し冷まします。
2. ご飯の中心にカレーを入れておにぎりを成形します。
3. 海苔を巻いて完成です。

### 2. キムチおにぎり

**材料（2個分）**:
- 温かいご飯 1合
- ビーガンキムチ 50g（魚介エキス不使用のもの）
- ごま油 小さじ1/2
- 白いりごま 小さじ1

**作り方**:
1. キムチを細かく刻みます。
2. ご飯にごま油を混ぜ、さらに刻んだキムチを混ぜます。
3. 手に塩水をつけ、おにぎりを成形します。
4. 白いりごまをまぶし、海苔を巻きます。

### 3. アボカドおにぎり

**材料（2個分）**:
- 温かいご飯 1合
- 完熟アボカド 1/2個
- レモン汁 小さじ1
- 塩 ひとつまみ
- 白いりごま 小さじ1

**作り方**:
1. アボカドをフォークで潰し、レモン汁と塩を混ぜます。
2. ご飯の中心にアボカドを入れておにぎりを成形します。
3. 白いりごまをまぶし、海苔を巻きます。

### 4. 五穀米のおにぎり

**材料（2個分）**:
- 白米 2/3合
- 雑穀ミックス 1/3合
- 塩 小さじ1/4
- 海苔 2枚

**作り方**:
1. 白米と雑穀ミックスを一緒に炊きます。
2. 炊きあがったご飯に塩を混ぜます。
3. 手に塩水をつけ、おにぎりを成形します。
4. 海苔を巻いて完成です。

### 5. 炊き込みひじきおにぎり

**材料（2個分）**:
- 米 1合
- 乾燥ひじき 10g
- にんじん 1/4本（細切り）
- 油揚げ（ビーガン対応の大豆製品） 1/2枚（細切り）
- 醤油 大さじ1
- みりん 大さじ1/2
- 塩 ひとつまみ

**作り方**:
1. ひじきを水で戻します。
2. 米を研ぎ、通常の水加減にします。
3. 戻したひじき、にんじん、油揚げ、調味料を加えて炊きます。
4. 炊きあがったら全体を混ぜ、おにぎりを成形します。
5. 海苔を巻いて完成です。

## 栄養バランスを考えたおにぎり

### 1. 豆腐と野菜のおにぎり

**材料（2個分）**:
- 温かいご飯 1合
- 木綿豆腐 1/4丁（水切りして細かく潰す）
- 小松菜 2〜3枚（茹でて細かく刻む）
- 人参 1/4本（細かく刻んで茹でる）
- 醤油 小さじ1
- 塩 ひとつまみ
- 白いりごま 小さじ1

**作り方**:
1. 潰した豆腐、刻んだ小松菜と人参を混ぜ、醤油と塩で味付けします。
2. ご飯に1の具材を混ぜます。
3. 手に塩水をつけ、おにぎりを成形します。
4. 白いりごまをまぶし、海苔を巻きます。

### 2. 玄米と豆のおにぎり

**材料（2個分）**:
- 玄米 1合
- 茹でた小豆または黒豆 大さじ2
- 塩 小さじ1/4
- 海苔 2枚

**作り方**:
1. 玄米を炊きます（圧力鍋を使うとふっくら炊けます）。
2. 炊きあがった玄米に茹でた豆と塩を混ぜます。
3. 手に塩水をつけ、おにぎりを成形します。
4. 海苔を巻いて完成です。

### 3. ひよこ豆とアボカドのおにぎり

**材料（2個分）**:
- 温かいご飯 1合
- 茹でたひよこ豆 1/4カップ（軽く潰す）
- アボカド 1/4個（潰す）
- レモン汁 小さじ1/2
- 塩 ひとつまみ
- クミン 少々（お好みで）
- 白いりごま 小さじ1

**作り方**:
1. ひよこ豆とアボカドを混ぜ、レモン汁、塩、クミンで味付けします。
2. ご飯の中心に1の具材を入れておにぎりを成形します。
3. 白いりごまをまぶし、海苔を巻きます。

## 保存とお弁当のコツ

### おにぎりの保存方法

1. **当日中に食べる場合**:
   - ラップで包んで冷蔵庫で保存します。
   - 海苔は食べる直前に巻くと、しっとりせずに美味しく食べられます。

2. **冷凍保存**:
   - おにぎりを個別にラップで包み、冷凍用保存袋に入れて冷凍します。
   - 解凍は自然解凍か、ラップを外して電子レンジで温めます。
   - 海苔は解凍後に巻くのがおすすめです。

### お弁当に入れるコツ

1. **衛生面**:
   - 具材は十分に加熱し、冷ましてから使用します。
   - 手や調理器具は清潔に保ちます。

2. **見た目の工夫**:
   - 海苔で模様をつけたり、ごまや青のりで色をつけたりして見た目を楽しくします。
   - 小さめに作って数種類詰めると、バラエティ豊かなお弁当になります。

3. **組み合わせ**:
   - おにぎりだけでなく、野菜のおかずも一緒に入れると栄養バランスが良くなります。
   - ビーガンのおかず例：きんぴらごぼう、ひじきの煮物、野菜の酢の物など。

## まとめ

ビーガンおにぎりは、動物性食品を使わなくても様々な具材や調理法で美味しく作ることができます。日本の伝統食であるおにぎりを、現代のビーガンライフスタイルに合わせてアレンジしてみてください。季節の野菜や豆類、海藻類を取り入れることで、栄養バランスも良く、満足感のある一品になります。

手軽に作れて持ち運びも便利なおにぎりは、忙しい日常の中での健康的な食事選択として最適です。ぜひお気に入りのレシピを見つけて、日々の食事に取り入れてみてください。
